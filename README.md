# Hex

基于 Java Swing 的 Hex 棋类游戏，清华大学谌卫军老师「Java 程序设计」课程大作业。

![游戏界面](pics/1.png)

## 游戏规则

Hex 棋是一种双人策略棋盘游戏。棋盘为 9×9 六边形网格，共 61 个有效格位。

1. 红方先行，双方交替落子
2. 选中己方棋子后，可选择两种移动方式：
   - **复制**（1 格范围，绿色高亮）：在目标位置生成棋子副本，原棋子保留
   - **跳跃**（2 格范围，橙色高亮）：棋子移至目标位置，原位置清空
3. 落子后，目标位置相邻的所有对方棋子被同化为己方颜色
4. 当轮到某方行棋时无合法走法，游戏结束，棋子数多者获胜
5. 每步限时 30 秒，超时判负

![高亮提示](pics/8.png)

## 功能特性

- 完整的六边形棋盘渲染与交互
- 选中棋子时高亮显示所有合法目标位置（绿色 = 复制，橙色 = 跳跃）
- 实时显示双方棋子数量与剩余时间
- 支持随时重新开始

## 运行方式

**直接运行（需要 JRE）：**

```bash
java -jar hex.jar
```

**从源码编译：**

```bash
javac -d out src/hex/*.java
java -cp out hex.GUI
```

## 项目结构

```
src/hex/
├── Params.java   # 全局参数：窗口尺寸、颜色方案、计时设置
├── Board.java    # 棋盘逻辑：Cell 棋格建模、邻域关系、同化机制
└── GUI.java      # 图形界面：棋盘面板绘制、信息面板、事件处理
```

- `hex.jar` — 预编译可执行文件
- `report.pdf` — 课程实验报告
- `project.pdf` — 课程项目说明文档
